# icebert-sentiment

A model to analyze the sentiment of Icelandic texts. Easy to train on PC.

To use it directly, please download the full model. Visit https://www.modelscope.cn/models/BrianWesley/icebert-sentiment-finetuned
and find icebert-sentiment-finetuned.zip
Then, download simple_icebert_sentiment.py in this repository.

The data used for training is from mbl.is, a famous news website in Iceland.

The model is upgraded recently.
Based on the evaluation code, its accuracy is more than 80%, making it closer to production.

Further improvement upcoming...
